# Overview

**DLite** (Deep Learning Inference Toolkit for Edge)는,  
얼굴을 감지하고 신원을 인식할 수 있는, 고성능, 고효율, 고가용한 AI 얼굴 인식 엔진을 제공하며,  
얼굴 인식 응용 프로그램을 개발하거나, 얼굴 인식을 기존 시스템에 통합하거나, 또는 기존 시스템의 기능을 확장할 수 있는 인터페이스(API)를 제공합니다.

## Core Engine
* DeviceSDK: Edge Device, Desktop 환경에서 사용하는 추론 엔진
* MobileSDK: Mobile 환경에서 사용하는 추론 엔진
* ServerSDK: 서버 환경에서 사용하는 (CPU, GPU) 추론 엔진
* MatchingSDK: 서버 환경에서 사용하는 고속의 Feature-Vector 매칭엔진
* X-RaySDK: X-Ray 수화물 사진에서 위험물질을 검출하는 추론 엔진
