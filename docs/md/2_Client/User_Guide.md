* Face Detection
* Face Feature-Extraction
* Face Matching
* Face Attribute (Sunglass, Hat)
* Face HeadPose (yaw/pitch/roll)
* Face Landmark Detection
* Covid Mask Detection
* Image Quality Check (Blur, Frontal)
* Liveness (Depth-Based)
